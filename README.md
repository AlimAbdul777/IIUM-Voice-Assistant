# IIUM-Voice-Assistant

## IIUM KOE Voice Assistant
[cite_start]Mira is an AI assistant designed for the IIUM Kulliyyah of Engineering[cite: 58].

## Features
- **STT:** OpenAI Whisper (via Groq) [cite: 11]
- **LLM:** Llama 3.1 8B [cite: 13]
- **TTS:** Microsoft Edge TTS [cite: 14]

## How to Run
1. Install dependencies: `pip install -r Requirements.txt`
2. Add your Groq API Key to `main.py`.
3. Run the script: `python main.py`
